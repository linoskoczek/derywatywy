## About

This program learns how to create female derivatives from Polish, male-form words:

`lekarz -> lekarka`

`doktor -> doktorka`

## Instructions to run

```
git clone https://github.com/linoskoczek/derywatywy.git
cd derywatywy
pip install -r requirements.txt
python3 projekt.py
```
